# Cátedra de Balances de Materia y Energía e Introducción a la Ingeniería Química - Facultad de Ciencias Exactas y Tecnología - Universidad Nacional de Tucumán

Este repositorio contiene el código fuente y los activos de la página web oficial de la cátedra. El sitio está diseñado para servir como plataforma educativa para los alumnos y como portal de vinculación tecnológica para servicios de consultoría externa.

## 🚀 Tecnologías Utilizadas
* **Hosting:** GitHub Pages
* **Framework:** Bootstrap 5 (HTML/CSS estático)
* **Cálculo Científico:** Math.js (Álgebra lineal) y PyScript (Termodinámica en Python)

## 📂 Estructura del Proyecto
* `/index.html` - Página de inicio e identidad institucional.
* `/equipo.html` - Staff docente, tesis y líneas de investigación.
* `/servicios.html` - Catálogo de servicios de optimización y modelado para la industria.
* `/alumnos/` - Repositorio de guías de trabajos prácticos y material didáctico.
* `/herramientas/` - Calculadoras interactivas y herramientas de simulación.
